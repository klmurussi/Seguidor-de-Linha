# Seguidor-de-Linha
Robô autonomo seguidor de linha da Titans

## Sobre
Um robô seguidor de linha, como o nome sugere, é um veículo guiado automaticamente, que segue uma linha visual embutida no chão ou teto. Normalmente, a linha é o caminho em que o robô vai e será uma linha preta em uma superfície branca, mas o outro caminho (linha branca em uma superfície preta) também é possível.

Robôs seguidores de linhas grandes são normalmente usados ​​em indústrias para auxiliar no processo de produção automatizado. Eles também são usados ​​em aplicações militares, fins de assistência humana, serviços de entrega, etc.
